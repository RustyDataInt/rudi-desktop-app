## RuDI Desktop App

The [Rusty Data Interface](https://rustydataint.github.io/) (RuDI) 
is a standardized framework for developing and running HPC data 
analysis **pipelines** and interactive visualization **apps**
with a Rust-first mindset.

This is the repository for the **RuDI Desktop App**, a cross-platform
[Electron](https://www.electronjs.org/) desktop app
that makes it easy to connect from your Windows or Mac PC 
to the computer where you will run the Dioxus app backend 
servers that feed data to your running data analysis apps. 

### Installation

Please download the relevant installer for your local operating system:
- [Download for Windows](https://github.com/RustyDataInt/rudi-desktop-app/releases/latest/download/rudi-desktop-app-Setup.exe)
