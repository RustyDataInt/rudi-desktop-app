## {{page.title}}

The RuDI Desktop is a cross-platform 
[Electron](https://www.electronjs.org/) desktop app
that makes it easy to connect from your Windows or Mac PC 
to the computer where you will run the Dioxus app backend 
servers that feed data to your running data analysis apps. 

### Install the RuDI Desktop

Download the relevant installer for your local operating system:

- {% include external-link.html href="https://github.com/RustyDataInt/rudi-desktop-app/releases/latest/download/rudi-desktop-app-Setup.exe" text="Download for Windows" %}
- {% include external-link.html href="https://github.com/RustyDataInt/rudi-desktop-app/releases/latest/download/rudi-desktop-app.dmg" text="Download for Mac" %}

You may be prompted to confirm the download, installation, or use of the RuDI Desktop. 
Please read the [security notes](security-notes) to understand the risks and benefits 
of using the Desktop.

The Desktop features an auto-update process, so once installed you should always have the most recent code.
